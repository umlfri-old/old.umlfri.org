How to get and install UML .FRI?

Download UML .FRI package from Downloads at first and unpack it into some your directory. If you have any linux distribution, run command $python main.py in UML.FRI directory (python, pygtk, gtk+... are required). For Windows, open bin folder and run uml_fri.exe.
