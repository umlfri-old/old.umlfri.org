UML .FRI

UML .FRI is visual diagram editor. It's aimed towards diagram types commonly used by people involved in IT. It's being developed by students at the Faculty of Management Science and Informatics, University of Zilina.

Features

Currently, it's possible to create and edit:

  • UML class diagram

  • UML object diagram

  • UML activity diagram

  • UML use case diagram

  • UML state diagram

  • Flow chart diagram

It's also possible to:

  • Export diagram as image in various formats (PNG, PDF, SVG, postscript)

  • Add your own diagram type without modifying application source code

Development

Project UML .FRI was started at the Faculty of Management Science and Informatics, University of Zilina in 2005. From the very beginning it has been being developed by students and the development is in fact the main scope of one of theirs subject. Normally, students are attending this subject for three semesters and during this period of time they have to understand the project, add something of their own and during the last semester, pass their knowledge to the next generation of students. This is why this project is called “generational”.

License

UML .FRI is released under GNU/GPL license. This means that anyone can freely copy and use it. Moreover, it's being distributed with source code, which can be modified and reused with condition, that every software that uses any part of UML .FRI source code must be as well released under GNU/GPL license.
