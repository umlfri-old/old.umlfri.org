Summary0000319: Pad aplikacie pri pokuse o zistenie lokalizacnych nastaveni pod Mac OS X
Description
Funkcia "getdefaultlocale()" modulu "locale", vracajuca lokalizacne nastavenia krajiny, kde je aplikacia spustana vo forme n-tice "(language code, encoding)" pri starte aplikacie pod Mac OS X (version10.5.6) vrati nasledujucu dvojicu hodnot:

(None,'mac-roman').

Kedze prva hodnota v ramci vratenej dvojice hodnot je tzv NoneType, pricom ocakavana hodnota ma byt dat. typu string, aplikacia pada pri pokuse vratit retazec hodnot vratenej n-tice zlozeny z jednotlivych poloziek n-tice oddelenych bodkou podla nasledovneho konstruktu:

return '.'.join(locale.getdefaultlocale())
TypeError: sequence item 0: expected string, NoneType found
