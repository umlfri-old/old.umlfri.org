Summary0000003: Refactoring XML subsystému
DescriptionTreba prerobiť celý XML subsystém.

Aktuálne je na parsovanie a prácu s XML databázami použitá knižnica minidom implementujúca subset w3c štandardu DOM. Ten je navrhovaný hlavne pre jazyky Java a JavaScript (ECMAScript). Zdrojový kód v Pythone teda nieje až taký prehľadný, nakoľko má diametrálne odlišnú syntax a spôsob objektového programovania.

V Pythone verzie 2.5 bola začlenená knižnica ElementTree implementujúca nový interface pre prístup ku XML orientovaný konkrétne na možnosti tohoto jazyka. Tento interface je použitý aj v knižnici lxml, ktorá ho ešte rozširuje o mnoho iných vecí (plná podpora dotazovacieho jazyka XPath, validácia voči XML schéme a podobne)

Takže odporúčam použiť na prácu s XML databázami knižnicu lxml a prevádzať aj validáciu voči XML schéme. Na systéme s Pythonom verzie min. 2.5 by mohla byť knižnica lxml nepovinná a teda by bola možnosť použiť ElementTree. V tom prípade ale musí vypísať varovanie na konzolu, že nebude fungovať validácia
